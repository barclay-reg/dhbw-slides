Übungen 1/2

  1. Spiele folgende Szenarien durch; nach jedem Szenario sollte git status folgenden Text ausgeben "nothing to commit, working tree clean", mit git-cat und ggf. git ls-tree sollen die Git-Objekte betrachtet werden

    1. Erstellen einer neuen Datei (inkl. Inhalt), Hinzufügen zum Index, danach diesen Committen

    2. Ändern einer bestehenden Datei (inkl. Inhalt), Hinzufügen zum Index, danach diesen Committen

    3. Hinzufügen einer neuen Datei (inkl. Inhalt) in einem Unterordner, Hinzufügen zum Index, Committen

Übungen 2/2

  • Fortsetzung Szenarien

    1. Umbenennen einer bestehenden Datei, Hinzufügen zum Index (Beides, die Löschung und die neue Datei), Committen

    2. Ändern einer bestehenden Datei, alte Version der Datei wieder herstellen

    3. Ändern einer bestehenden Datei, Hinzufügen zum Index (nicht committen), Entfernen aus dem Index, Datei nochmals ändern, Hinzufügen zum Index, Committen

    4. Ändern einer bestehenden Datei, Hinzufügen zum Index (nicht committen), Datei nochmals ändern, abermals Hinzufügen zum Index, Committen
