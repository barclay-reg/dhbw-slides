=== Java, die Sprache

* Syntax des Codes
* Klammern, Semikolon, Methodenaufrufen, Klassendefinition, ...
----
<Object-Instanz>.<methode>(<MethodenSignatur>)
// Kommentar
----

Beispiel

[source, java]
----
String name = null;
name = "Robert":
name.substring(2,5) // "ber"
----
